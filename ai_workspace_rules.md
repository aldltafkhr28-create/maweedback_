# تعليمات الذكاء الاصطناعي لمشروع (Maweed)
هذا الملف يحتوي على السياق والمهارات التي تعلمها المساعد الذكي لخدمة هذا المشروع بأفضل شكل.

## التقنيات المستخدمة (Tech Stack)
- لغة البرمجة: Java
- إطار العمل: Spring Boot
- قاعدة البيانات: PostgreSQL

## المهارات والتعليمات الثابتة (AI Skills & Rules)
1. كتابة أكواد Java نظيفة ومنظمة وفقاً لمعايير Spring Boot (Controllers, Services, Repositories).
2. عند كتابة أكواد خاصة بقاعدة البيانات، يتم استخدام إعدادات تناسب PostgreSQL وتجنب الأكواد الخاصة بقواعد البيانات الأخرى.
3. شرح الأكواد المعقدة للمستخدم بأسلوب مبسط ومفهوم باللغة العربية (اللهجة المصرية) لأن المستخدم في مرحلة التعلم وبناء المشروع.
4. التأكد من التعامل مع العلاقات (Relationships) في قاعدة البيانات باستخدام (JPA/Hibernate) بشكل صحيح لتجنب مشاكل الأداء.
